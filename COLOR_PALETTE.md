# Data Dumpster Diver - Dark Mode Color Palette

## Overview
A modern, accessible dark mode color palette designed for data visualization and conversation exploration. The palette emphasizes readability, visual hierarchy, and professional aesthetics while maintaining an approachable feel.

## Color System

### Primary Colors
- **#3B82F6** (Primary Blue) - Main actions, navigation, links, and primary interactive elements
- **#2563EB** (Primary Blue Hover) - Hover states for primary elements
- **#60A5FA** (Primary Blue Light) - Accents, highlights, and subtle emphasis

### Secondary Colors
- **#8B5CF6** (Secondary Purple) - Secondary actions, alternative highlights
- **#7C3AED** (Secondary Purple Hover) - Hover states for secondary elements

### Accent Colors
- **#06B6D4** (Accent Cyan) - Data visualization, charts, graphs
- **#10B981** (Accent Green) - Success states, completed uploads, positive indicators
- **#F59E0B** (Accent Orange) - Warnings, attention-grabbing elements
- **#EF4444** (Accent Red) - Error states, destructive actions, failed operations

### Neutral Grays (Dark Theme Foundation)
- **#0F172A** (Background Primary) - Main application background
- **#1E293B** (Background Secondary) - Cards, panels, content areas
- **#334155** (Background Tertiary) - Hover states, elevated surfaces
- **#475569** (Disabled) - Disabled interactive elements
- **#64748B** (Chart Axis) - Chart axis labels and secondary data
- **#94A3B8** (Text Muted) - Placeholder text, subtle information
- **#CBD5E1** (Text Secondary) - Secondary content, descriptions
- **#F8FAFC** (Text Primary) - Primary text content
- **#FFFFFF** (Text Inverse) - Text on colored backgrounds

### Semantic Colors
- **#1E293B** (Surface) - Input fields, dropdown backgrounds
- **#334155** (Border) - Borders, dividers, separators
- **#3B82F640** (Focus Ring) - Blue with 25% opacity for accessibility focus states

### Data Visualization Specific
- **#3B82F6** (Chart Line) - Primary chart lines and data points
- **#3B82F620** (Chart Fill) - Chart area fills with 12% opacity
- **#334155** (Chart Grid) - Chart grid lines and reference lines
- **#64748B** (Chart Axis) - Chart axis labels and scales

### Conversation View Colors
- **#3B82F6** (Message User BG) - User message backgrounds
- **#FFFFFF** (Message User Text) - User message text
- **#1E293B** (Message Assistant BG) - Assistant message backgrounds
- **#F8FAFC** (Message Assistant Text) - Assistant message text
- **#374151** (Message System BG) - System notification backgrounds
- **#CBD5E1** (Message System Text) - System notification text

## Design Rationale

### Accessibility
- All text combinations meet WCAG AA contrast ratios (4.5:1 or higher)
- Focus states use semi-transparent overlays for clear visibility
- Color combinations tested for color blindness accessibility

### Visual Hierarchy
- Primary blue draws attention to key actions and navigation
- Purple provides visual interest without overwhelming the interface
- Neutral grays create depth and separation between content areas
- Accent colors provide clear semantic meaning (success, warning, error)

### Data Visualization
- Cyan stands out against dark backgrounds for charts and graphs
- Consistent opacity levels (20%, 40%) for overlays and fills
- Grid lines use subtle grays to avoid competing with data

### Application to Pages

#### Upload Page
- Clean, minimal interface using primary grays
- Green accent for successful uploads
- Blue primary action for upload button
- Orange for warnings about file formats

#### Conversations List
- Search interface with subtle borders
- Hover states using tertiary background
- Purple accents for highlighting selected items
- Clear text hierarchy for conversation titles

#### Conversation View
- Distinct message bubble colors for user vs assistant
- System messages using muted grays
- Timestamps and metadata using secondary text
- Interactive elements using primary blue

## Usage Guidelines
1. Use primary blue for main interactive elements
2. Reserve accent colors for specific semantic meaning
3. Maintain consistent opacity levels for overlays
4. Ensure text remains readable at all sizes
5. Test color combinations in different lighting conditions

This palette provides a solid foundation for a modern, accessible dark mode interface that works well for data exploration and conversation management.
